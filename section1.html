Image generators: What are they? How do they work?

Image generators use user's text descriptions or reference imagery to create a new image. By accessing millions of images and artworks online (using these machine learning models called neural networks), image generators are able to reference this source material, learning their patterns and consistent structures, they're able to combine multiple aspects of these sources to create an image reflecting the user's prompt.
